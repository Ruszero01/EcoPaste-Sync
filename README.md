## 🌟 分支说明

> 📋 **本分支基于官方 EcoPaste v0.5.0 版本，提供云同步功能的临时解决方案，由于官方云同步功能迟迟未上线🥲。**

### ✨ 当前特性

- 🔄 **WebDAV 云同步**：通过 WebDAV 协议实现剪贴板数据的多设备同步
- 🗂️ **智能同步模式**：支持多种同步策略，包括按内容类型、收藏状态等条件进行选择性同步
- 🔐 **数据安全保障**：本地存储为主，云端同步为辅的混合模式，数据完全可控
- ⚡ **实时同步引擎**：智能冲突检测与解决，支持双向同步和增量更新

### 🚀 未来规划

- 📌 **多协议云存储支持**：支持 OneDrive、Google Drive、Dropbox 等更多云存储协议
- ⚡ **云数据库实时同步**：基于云数据库的实时同步功能，多设备间即时更新

### 📦 使用说明

本分支主要面向急需跨设备同步功能的用户，作为官方云同步功能发布前的临时解决方案。如果您只需要本地剪贴板管理功能，建议使用 [官方主分支](https://github.com/EcoPasteHub/EcoPaste)。

## 📥 获取应用程序

> 💡 **本分支专注于云同步功能开发，如需下载完整应用程序，请访问官方主分支**

### 🔗 访问官方主分支

- 🌐 **GitHub 主页**：[EcoPasteHub/EcoPaste](https://github.com/EcoPasteHub/EcoPaste)
- 📱 **官方下载**：[Releases 页面](https://github.com/EcoPasteHub/EcoPaste/releases)
- 📚 **使用文档**：[EcoPaste 官网](https://ecopaste.cn/)

### 🛠️ 从源码构建（开发版）

```bash
# 克隆本分支
git clone -b add-sync https://github.com/EcoPasteHub/EcoPaste.git
cd EcoPaste

# 安装依赖
pnpm install

# 开发模式运行
pnpm tauri dev

# 构建生产版本
pnpm tauri build
```

> ⚠️ **注意**：本分支为开发分支，可能包含实验性功能。生产使用建议选择官方稳定版本。

## 核心功能

### 📋 剪贴板管理
- **多格式支持**：支持纯文本、富文本、HTML、图片、文件等多种剪贴板格式
- **智能去重**：自动识别重复内容，避免冗余存储
- **历史记录**：完整保存剪贴板历史，支持按时间、类型、收藏状态筛选
- **快速搜索**：强大的搜索功能，快速定位所需内容
- **分组管理**：按内容类型自动分组（文本、图片、文件等）
- **收藏系统**：标记重要内容为收藏，永久保存不被清理

### 🎨 界面与交互
- **现代化设计**：基于 Ant Design 的美观界面，支持亮色/暗色主题
- **多种窗口模式**：浮动窗口、停靠窗口等多种显示方式
- **快捷操作**：一键复制、粘贴、删除等快捷操作
- **快捷键支持**：全局快捷键快速唤出，快速粘贴键（1-9）高效操作

### 🔧 系统集成
- **多平台支持**：Windows、macOS、Linux（x11）全平台兼容
- **开机自启**：支持开机自动启动，后台静默运行
- **系统托盘**：最小化到系统托盘，不占用任务栏空间
- **全局热键**：自定义全局快捷键，随时快速访问

### 📊 数据管理
- **本地数据库**：基于 SQLite 的可靠本地存储
- **自动清理**：可配置的历史记录自动清理策略
- **数据导出**：支持数据备份和导出功能
- **容量管理**：智能管理存储空间，避免过度占用
- **数据恢复**：支持从备份文件恢复数据

### ☁️ 云同步功能（本分支特色）
- **WebDAV 同步**：支持通过 WebDAV 协议进行多设备数据同步
- **智能同步模式**：多种同步策略，支持按内容类型、收藏状态等条件选择
- **冲突解决**：智能冲突检测与自动解决机制
- **双向同步**：支持多设备间的双向数据同步

### ⚙️ 个性化设置
- **主题定制**：亮色/暗色主题，跟随系统或手动切换
- **语言支持**：中文简体、繁体、英文、日文等多语言
- **行为配置**：丰富的行为选项，满足不同使用习惯
- **性能优化**：可配置的性能参数，平衡功能与资源占用
- **更新管理**：自动检查更新，支持增量更新

### 🔒 隐私与安全
- **本地优先**：数据主要存储在本地，完全可控
- **隐私保护**：敏感数据不上传，或仅在用户授权下同步
- **权限管理**：最小权限原则，仅请求必要的系统权限
- **安全审计**：代码开源透明，接受社区安全审查

## 📖 使用指南

### 快速开始
1. **下载安装**：从 [Releases](https://github.com/EcoPasteHub/EcoPaste/releases) 页面下载对应平台的安装包
2. **启动应用**：首次启动会自动开始监听剪贴板
3. **配置设置**：通过偏好设置自定义行为和外观
4. **开始使用**：使用快捷键唤出界面，开始管理剪贴板

### 云同步配置（本分支）
1. **准备 WebDAV 服务**：确保你有可用的 WebDAV 服务
2. **配置连接**：在偏好设置的"云同步"中填写服务器信息并测试连接
3. **开始同步**：选择合适的同步策略并开始同步

## 问题反馈

1. 🔍 优先查阅[常见问题](https://ecopaste.cn/problem/macos/damage)或浏览已有 [issues](https://github.com/EcoPasteHub/EcoPaste/issues)。

2. ❓ 如果问题仍未解决，请提交新的 [issue](https://github.com/EcoPasteHub/EcoPaste/issues/new/choose)，并附上详细描述，方便我们快速定位和解决。
