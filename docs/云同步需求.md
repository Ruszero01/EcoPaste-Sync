# 云同步需求

## 目的

为 EcoPaste 添加云同步功能，实现在终端A复制后，在终端B可以立刻看到复制的内容

## 需求

- 使用 webdav 方式同步
- 实时同步，当数据库变更时同步上传到服务器
- 多端实时刷新最新数据
- 云端没有同步路径时自动创建，已有路径时直接覆盖
